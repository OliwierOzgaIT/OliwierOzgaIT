### 🔭 [Azure Serverless Site Monitor](https://github.com/OliwierOzgaIT/azure-serverless-monitoring)
Fully serverless uptime monitoring platform — 12 sites checked every 5 minutes via Python Azure Functions. Results stored in Table Storage, exposed via REST API, displayed on a live static dashboard hosted on Blob Storage. Alerts via Logic Apps. All infrastructure provisioned with modular Terraform.

### 🧪 [Chaos Engineering & Resilience Lab](https://github.com/OliwierOzgaIT/chaos-engineering-resilience-testing)
Fault injection lab validating system resilience across three layers — CPU saturation (stress-ng, 99.8%), network latency (Pumba, 3000ms), and storage exhaustion (fallocate, 95%). Deployed on Azure with Terraform, observed with Azure Monitor and Linux native telemetry.

`Azure VM` `Docker` `Pumba` `stress-ng` `Terraform` `Azure Monitor`
